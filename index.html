Caso de Estudio: Mercado Libre

Implementación de CI/CD con GitHub Actions y GitHub Pages

Introducción

Mercado Libre es la plataforma de comercio electrónico más grande de América Latina, ofreciendo servicios de marketplace, pagos digitales (Mercado Pago) y logística (Mercado Envíos).

Problema Identificado

Debido a su alto volumen de usuarios y despliegues frecuentes, Mercado Libre requiere procesos automatizados que garanticen calidad, rapidez y estabilidad en cada cambio realizado sobre sus aplicaciones web.

Solución Propuesta

La adopción de prácticas de Integración Continua (CI) y Despliegue Continuo (CD) permite validar automáticamente los cambios, reducir errores humanos y asegurar entregas constantes y confiables.

Aplicación en este Proyecto

En este taller se implementa un pipeline CI/CD usando GitHub Actions, donde cada cambio enviado al repositorio es integrado y desplegado automáticamente en GitHub Pages, simulando un flujo similar al utilizado por plataformas como Mercado Libre.
